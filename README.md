# job-listings-board

## Overview

This project is a responsive Job Listings Board built with React and Tailwind CSS. The application fetches job data from an external API and allows users to filter listings using job attributes such as role, level, languages, and tools.

## Features

- Fetch job listings from API
- Loading and error states
- Responsive layout (mobile and desktop)
- Skill tag filtering
- Multiple filters support
- Clear filters functionality
- Interactive hover effects

## Technologies Used

- React
- Tailwind CSS
- JavaScript
- Vite

## API

Job listings are fetched from:

https://jobs-api-l3e2.onrender.com/api/jobs

## Installation

Clone the repository:

git clone https://github.com/osemwengiearit/job-listings-board.git
Navigate to project folder:

cd job-listings-board

Install dependencies:

npm install

Run development server:

npm run dev
